# MentionedMeme
AI generated images of the famous "Brazil Mentioned" meme of K-On anime. Goal is to only generate small fractions of the image (bottle, cap, background, dress). Not regenerate the entire image through an AI via a prompt to some diffusion model. This way the memes' "comedy" element is better retained.


# start up
use command for setting start up
`uv venv`
`uv pip install requirements.txt`
